JY.attr(elem,attrName,attrValue)返回值:JY|string

概述

根据所提供的ID，获取或设置DOM的属性值

参数

elemstring

要查找的DOM节点的ID

attrNamestring

对应的DOM属性名称

attrValuestring

对应的DOM属性的值，无此参数则是返回属性值

示例

HTML代码:

<div><span id="sp_id"></span></div>

JY代码:

JY.attr("sp_id","name","spname")

结果

<span id="sp_id" name="spname"></span>
